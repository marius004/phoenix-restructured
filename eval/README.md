Code Execution Enviroment(RCE)
==== 

### Requirements: 
* To set up the evaluator you need to be on Linux!
* https://github.com/marius004/isolate

### Installation: 
```
sudo apt-get update
sudo apt-get install libcap-dev asciidoc-base
cd ~/Downloads
git clone https://github.com/marius004/isolate
cd isolate
make
sudo make install
```
